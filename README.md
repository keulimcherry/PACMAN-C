# PAC-MAN Raylib Clone

Authentic Pac-Man clone implemented in C using raylib.

Features
- Playable Pac-Man with ghost AI, power pellets, fruit, levels, and scoring
- Portable code designed to work with raylib

Prerequisites
- raylib (tested with v5.x)
- C compiler (MSVC / MinGW) or Visual Studio

Repository layout
- `PACMANRAY/main.c` — game source
- `assets/` — spritesheet and audio files (not included in this repository)
- `README.md`, `ASSETS.md`, `LICENSE`, `CONTRIBUTING.md`, `docs/`

Quick start
1. Install raylib on your platform (see raylib documentation).
2. Build the project with your compiler and link to raylib.
3. Create an `assets/` folder next to the executable containing required sprite and audio files (see `ASSETS.md`).

Notes
- Docs and markdown files in this repository are saved in UTF-8 encoding to be compatible with GitHub Pages (Jekyll).
- If you see an encoding error from GitHub Pages, convert your files to UTF-8 and remove non-standard characters.
